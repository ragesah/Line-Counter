The code is available in "code" folder. It contains:
1. "main_script.py" or "main_script.ipynb" can be invoked to run the code. Jupyter Notebook has temporary result displayed.
2. "dir" folder used for code unit testing.
3. "utils" has helper scripts.


The existing code is designed to work only for ".txt" files sepecially, for average line computations. 

The code is tested to read through nexted directories starting from directory "dir":
1) directory with single file (both .txt and other)
2) directory with single file (only .txt)
3) directory with single file (only other extention)
4) directory with combination of files and folders
